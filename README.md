# aipaca
AI agents to work with ALP-aca, ALPs and axions

## Using the MCP in VSCode

Clone the repository and initialize the directory with poetry. In VSCode, open the file `.vscode/mcp.json`, and you should see the server `"AIPaca MCP server"`, and right about it, in light grey, a button to start it. Click it, and it should start and you can use it in the chat window.
